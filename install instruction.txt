    # Builiding detectron2 from source
    Check and load GCC with the correct version:
    gcc --version / g++ --version to check version
    module spider GCC (to check available GCC versions) 
    module load GCC/8.3.0 (or load any GCC version satisfying requirement https://detectron2.readthedocs.io/en/latest/tutorials/install.html#requirements)
    Build detectron2

    python -m pip install 'git+https://github.com/facebookresearch/detectron2.git' (https://detectron2.readthedocs.io/en/latest/tutorials/install.html#build-detectron2-from-source)
    pip install opencv-python
